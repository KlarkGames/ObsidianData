# Условная вероятность
#Теория_вероятности 

Условная вероятность - вероятность события B при условии, что случилось событие A ($P(A) > 0$)
$$P(B|A) = \frac{\#(A\cap B)}{\#A} = \frac{\#(A\cap B) / \#\Omega}{\#A / \#\Omega} = \frac{P(A\cap B)}{P(A)}$$
